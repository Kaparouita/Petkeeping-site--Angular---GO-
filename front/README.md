## 359 Frontend

This is the frontend for the 359 project regarding a pet keeping app.

### Presquites

- [Node.js](https://nodejs.org/en/)
- [Angular CLI](https://cli.angular.io/)

### Installation

1. ```npm install```
2. ```ng serve```

### Usage

1. Open [http://localhost:4200](http://localhost:4200) in your browser.
